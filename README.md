# Basic Chat Application using WebSockets
This project is a real-time chat application that allows multiple users to send and receive messages instantly. It is built using WebSockets, enabling a fast, low-latency communication channel between clients and server.

The application supports private and group messaging, making it a simple foundation for chat systems in web applications.

**Features:-**

* Real-time messaging using WebSockets

* Support for multiple users connected simultaneously

* Private and group chat support

* Instant message delivery without page reload

* Lightweight and simple interface for easy interaction

**Technology Stack:-**

* Frontend: HTML, CSS, JavaScript

* Backend: Node.js with socket.io library for WebSockets

**How to Run:-**

*1. Install backend dependencies:*
````
cd backend
npm install
````

*2. Start the WebSocket server:*
````
node server.js
````
